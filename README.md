# スクーミー サンプルコード：湿度＋温度センサーで7セグ表示

## 概要

光センサー（明るさ・湿度センサー相当）と温度センサーを組み合わせて、
条件に応じて7セグメントディスプレイに数値を表示するプログラムです。

- 土壌水分センサー（またはアナログセンサー）の値が50%を超えている
- かつ温度が30℃を超えている

上記2つの条件が同時に満たされたとき、7セグに `1111` を表示します。
どちらか一方でも満たされていない場合は `0` を表示します。

---

## 使用機材

| 機材 | 詳細 |
|------|------|
| スクーミーボード | オレンジボード（ATmega328pb） |
| アナログセンサー | 右上ポート接続（A1ピン使用） |
| 温度センサー | DS18B20（1-Wireプロトコル） |
| 7セグメントディスプレイ | TM1637（左上ポート接続） |

# スクーミー サンプルコード：土壌水分センサー＋温度センサーで条件判定

## 概要

土壌水分センサーと温度センサーを組み合わせて、
両方の条件が満たされたときに7セグメントディスプレイへ数値を表示するプログラムです。

---

## 使用機材

| 機材 | ポート |
|------|--------|
| スクーミーボード（オレンジ） | ― |
| 土壌水分センサー（アナログ） | 右上ポート |
| 温度センサー（DS18B20） | 左下ポート |
| 7セグメントディスプレイ | 左上ポート |

---

## 動作の流れ

ループ処理で以下の2段階の条件判定を行います。

```
ループ処理:
  1. 土壌水分センサーの割合 > 50%?
     ├─ YES → 2. 温度 > 30℃?
     │         ├─ YES → 7セグに「1111」を表示
     │         └─ NO  → 7セグに「0」を表示
     └─ NO  → 7セグに「0」を表示
```

両方の条件が揃ったときだけ `1111` が表示されます。
どちらか一方でも外れると `0` に戻ります
