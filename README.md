## Schoomy_project

## これは何？

スクーミー（オレンジボード）を使って、土壌水分と温度を同時に監視して、熱中症対策を促進するアプリ。
水分量と温度の両方が一定の基準を超えたとき、7セグメントディスプレイに数値を表示して知らせます。

---

## 動作内容

- 右上ポートの土壌水分センサーで、水分量を0〜100%の割合で読み取ります
- 左下ポートの温度センサー（DS18B20）で、現在の温度（℃）を読み取ります
- 以下の条件で7セグメントディスプレイの表示が変わります

```
ループ処理:
  1. 土壌水分センサーの割合 > 50%?
     ├─ YES → 2. 温度 > 30℃?
     │         ├─ YES → 7セグに「1111」を表示
     │         └─ NO  → 7セグに「0」を表示
     └─ NO  → 7セグに「0」を表示
```

---

## 使い方

1. スクーミーfox. でこのプロジェクトを開く
2. オレンジボードをUSBでパソコンに接続する
3. 「スクーミーボードへ書き込み」ボタンを押す
4. 書き込みが完了したら、センサーに触れたり温めたりして動作を確認する
